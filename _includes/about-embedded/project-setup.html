Set up your project

No more copy-pasting or ad-hoc regexes

Step 1

Configure file project

Go to i18n plugin Project configurations > Add setting profile > Select language files properties.

Step 2

Find hardcoded strings

Main menu, select Tools and select “Run i18n Ally inspections” to find and extract hardcoded strings.

Step 3

Extract strings

In inspection results window, extract strings one by one, the whole file or folder with one button click.
